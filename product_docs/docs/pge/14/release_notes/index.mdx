## EDB Postgres Extended Server 14.12.0

Release date: 2024-05-09

This is a minor release of EDB Postgres Extended Server 14. This release contains a variety of improvements over PostgreSQL 14.11. For more information about new features in the core 14.11 release, see [Postgres 14 release notes](https://www.postgresql.org/docs/14/release.html) .


### Migration

A dump/restore using [pg_dumpall](https://www.postgresql.org/docs/14/app-pg-dumpall.html) is required for those wishing to migrate data from any previous community PostgreSQL release.


### Changes


- Includes all changes up to version 14.12 of the community PostgreSQL code base.

- Includes all changes up to 2ndQuadrant Postgres version 14.12.0.
